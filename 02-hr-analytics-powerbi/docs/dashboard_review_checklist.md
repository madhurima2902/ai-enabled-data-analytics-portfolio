# Final Dashboard Review Checklist

Before publishing or sharing the dashboard:

## General

- [ ] Only final pages are visible: Overview, Demographics, Performance Tracker, Attrition Analysis.
- [ ] Old course-build pages are deleted or hidden.
- [ ] Navigation shows only the final pages.
- [ ] Header titles match page purpose.
- [ ] Subtitles match page purpose.
- [ ] No raw measure names appear in chart titles or legends.
- [ ] Visual spacing is consistent.
- [ ] Page background, header, navigator, and card styles are consistent.

## Overview

- [ ] KPI cards show exact counts, not `1K`.
- [ ] Hiring trend uses Active/Inactive labels instead of Yes/No.
- [ ] Department and job role visual is readable.
- [ ] Executive summary is concise.

## Demographics

- [ ] Page title is Workforce Demographics.
- [ ] Age bins are sorted logically.
- [ ] Age group label is `<20`, not `<=20`, if age 20 is included in `20-29`.
- [ ] Page is not accidentally filtered to Active or Inactive.
- [ ] Salary/ethnicity insight is worded responsibly.

## Performance Tracker

- [ ] Employee slicer is dropdown with search.
- [ ] Employee names have spaces.
- [ ] Cards show `Select employee` when no employee is selected.
- [ ] Inactive employees show next review as `Not applicable`.
- [ ] Chart Y-axis is fixed from 1 to 5.
- [ ] Review Date is formatted as short date in the table.

## Attrition Analysis

- [ ] Subtitle is attrition-specific.
- [ ] Overtime chart title is correct.
- [ ] Category charts use `% Attrition Rate`, not `% Attrition Rate Date`.
- [ ] Hire-year chart uses `% Attrition Rate Date`.
- [ ] Tenure bins are sorted correctly.
- [ ] Matrix shows 0 instead of blanks where appropriate.
- [ ] Key insight avoids causal overclaiming.
